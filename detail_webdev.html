Web Development

Belajar membuat website dari dasar hingga mahir dengan berbagai teknologi terkini. Program ini dirancang untuk memberikan keterampilan dan pengetahuan yang dibutuhkan untuk bekerja sebagai web developer. Berikut adalah materi-materi yang akan dipelajari:

1. HTML & CSS: Dasar-dasar pembuatan struktur halaman web menggunakan HTML dan styling menggunakan CSS.

2. Javascript: Pengenalan Javascript sebagai bahasa pemrograman utama untuk menambah interaktivitas pada situs web.

3. Front-End Frameworks: Pembelajaran tentang framework populer seperti Bootstrap untuk mempercepat pengembangan antarmuka pengguna yang responsif.

4. Back-End Development: Pengenalan ke bahasa pemrograman back-end seperti Node.js, Python (dengan Django atau Flask), Ruby on Rails, atau PHP. Penggunaan database (SQL, MySQL) dan bagaimana mengintegrasikannya dengan aplikasi web.

5. API & Web Services: Pemahaman tentang pembuatan dan konsumsi API (Application Programming Interface), serta integrasi dengan layanan web lainnya.

6. Web Hosting & Deployment: Cara mempublikasikan situs web secara online menggunakan layanan hosting, serta teknik deployment agar situs dapat diakses oleh pengguna.
